# 使lastUpdated参数生效的正确操作步骤：

  1. 首先确保博客项目的文件夹是一个基于git的文件夹
  2. 然后将.md文件add、commit并push(理论上说可以暂时不push)
  3. 最后执行npx vuepress build docs生成dist文件夹

# 备注

顺序很重要，必须先commit后再build才能使lastUpdated生效，这一点官方文档没有说清楚。。。
